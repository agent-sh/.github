# Contributing to agent-sh

Thanks for your interest in contributing. All repos under agent-sh follow these guidelines.

## Getting Started

1. Fork the repo
2. Create a feature branch (`git checkout -b feature/my-change`)
3. Make your changes
4. Run tests (`npm test` or equivalent)
5. Commit with conventional commits (`feat:`, `fix:`, `docs:`, `chore:`)
6. Open a PR against `main`

## AI-Assisted Contributions

AI-assisted contributions are welcome. You are responsible for all code you submit, regardless of how it was generated. Low-quality or unreviewed AI content will be closed.

## Code of Conduct

Be respectful. Be constructive. Focus on the work.

## Security

Report vulnerabilities via [SECURITY.md](https://github.com/agent-sh/.github/blob/main/SECURITY.md), not public issues.
